# Mobile Navigation

## Navigation Principles

- Role-specific tabs after login.
- Workout tracking should be reachable quickly.
- Active workout state should be recoverable from anywhere.
- Notifications should deep link to related content.

## Athlete Navigation

```text
Login/Register
  -> Athlete Tabs
      Dashboard
      Program
      Start Workout
      Progress
      Profile
```

## Trainer Navigation

```text
Login/Register
  -> Trainer Tabs
      Dashboard
      Athletes
      Programs
      Notifications
      Profile
```

## Admin Navigation

```text
Login/Register
  -> Admin Tabs
      Users
      Exercises
      Reports
      Logs
      Profile
```

## Deep Links

- Notification to program update
- Notification to workout completed detail
- Notification to trainer note
- Notification to relationship request
