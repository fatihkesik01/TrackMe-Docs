# TrackMe Product Backlog

## Epic: Core Coaching

### Feature: Program Builder

User Story:

- As a trainer, I want to create structured programs for athletes.
- As an athlete, I want to create self-guided programs.

### Feature: Workout Mode

User Story:

- As an athlete, I want to complete workouts set by set.
- As a trainer, I want athlete workout data to be recorded accurately.

### Feature: Templates

User Story:

- As a trainer, I want to reuse common training structures.

### Feature: Public Program Copying

User Story:

- As a user, I want to copy public programs.
- As a trainer, I want to assign copied programs to my athletes.
- As a user, I want copied public programs to stay stable even if the original program changes.
- As the system, I want copied programs to keep source program metadata as a Program Fork.

## Epic: Media

### Feature: MediaAsset

User Story:

- As the system, I need one media model for images, videos and audio.

### Feature: Avatar Photo

User Story:

- As a user, I want to upload an avatar photo for my profile.
- As the system, I want avatar photos to use the shared MediaAsset model.

### Feature: Cover Photo

User Story:

- As a user, I want to upload a cover photo for my profile.
- As the system, I want cover photos to use the shared MediaAsset model.

### Feature: Progress Photos

User Story:

- As an athlete, I want to upload progress photos.
- As an athlete, I want to view progress photos in a timeline.
- As an athlete, I want to compare before/after photos.
- As an athlete, I want to control progress photo visibility.
- As a trainer, I want to view shared progress photos of my athlete.

### Feature: Media Reporting

User Story:

- As a user, I want to report inappropriate or unsafe media.
- As an admin, I want to review reported media and make moderation decisions.

### Feature: Submission Videos

User Story:

- As an athlete, I want to send form videos to my trainer.

### Feature: Feedback Videos

User Story:

- As a trainer, I want to send video feedback to my athlete.

### Feature: Audio Feedback

User Story:

- As a trainer, I want to send voice feedback.

### Feature: Exercise Videos

User Story:

- As an athlete, I want to watch exercise demonstrations.
- As a user, I want to upload videos for exercises I created.
- As a trainer, I want to attach videos to exercises in my programs.
- As the system, I want to support future official/community exercise video distinction.

### Feature: Program Media

User Story:

- As a trainer, I want to attach video/image explanations to programs.
