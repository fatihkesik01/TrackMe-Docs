# Product Vision

TrackMe is not a simple gym logging app.

The goal is to create a professional training management system where:

- Trainers manage athletes professionally.
- Athletes track workouts efficiently.
- Workout history becomes meaningful.
- Performance data becomes measurable.
- RPE and fatigue can be monitored.
- Progress can be analyzed visually.
- Communication between trainer and athlete stays organized.

## Target Users

### Trainers

Trainers need an operational tool for managing many athletes, creating programs, checking compliance, reviewing history, and identifying risk signals such as excessive RPE or missed workouts.

### Athletes

Athletes need a fast and low-friction training experience. They should be able to start a workout, log sets, track rest, enter RPE, and finish without fighting the interface.

### Admins

Admins need control over user quality, exercise consistency, reporting, logs, and permissions.

## Product Personality

TrackMe should feel:

- Professional
- Fast
- Clean
- Reliable
- Modern
- Performance-focused

## Non-Goals

- TrackMe should not start as an AI workout generator.
- TrackMe should not prioritize social media features over training workflows.
- TrackMe should not rely on unstructured notes as the primary data model.
- TrackMe should not allow inconsistent exercise naming to damage analytics quality.
