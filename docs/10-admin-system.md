# Admin System

The admin system exists to maintain platform quality, data consistency, and operational safety.

## Admin Responsibilities

- Manage users
- Manage trainer accounts
- Manage athlete accounts
- Manage exercise library
- Prevent duplicate exercises
- Review reports
- Monitor logs
- Manage permissions
- Manage notifications

## Admin Screens

- User list
- User detail
- Trainer management
- Athlete management
- Exercise management
- Reports
- Logs
- Notification management
- Permission management

## Admin Backend Capabilities

- Search users
- Activate or deactivate users
- Assign or revoke roles
- Create and update exercises
- Review duplicate exercise candidates
- View system logs
- Send platform notifications
- Review relationship reports

## Admin Guardrails

- Admin actions should be audited.
- Dangerous actions should require confirmation.
- Hard delete should be avoided for business records.
- Admin endpoints should be rate limited and logged.
