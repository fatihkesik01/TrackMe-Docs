# TrackMe Roadmap

## P0 - Core Coaching Reliability

### Amac

Trainer-athlete coaching akisini guvenilir, net ve tekrar kullanilabilir hale getirmek.

### Basari Kriteri

Athlete programi sorunsuz uygular; trainer program yazar, takip eder ve feedback verebilir.

### Ozellikler

- Program Builder iyilestirme
- Workout Mode iyilestirme
- Template guvenilirligi
- Public program kopyalama
- Core auth/role kararliligi
- Progress tracking temel akisi

### Riskler

- Role karmasasi
- Program ownership belirsizligi
- Workout UX'in mobil icin yetersiz kalmasi

## P1 - Media-Enabled Coaching

### Amac

Video, fotograf ve ses ile gercek coaching feedback dongusu kurmak.

### Basari Kriteri

Athlete video/fotograf gonderir; trainer feedback verir; media guvenli saklanir ve dogru kisilere gorunur.

### Ozellikler

- MediaAsset altyapisi
- Avatar photo
- Cover photo
- Progress photos
- Progress photo timeline
- Before/after comparison
- Media reporting
- Admin media moderation
- Exercise videos
- Program videos
- Athlete submission videos
- Trainer feedback videos
- Audio feedback

### Riskler

- Medya maliyeti
- Upload performansi
- Privacy hatalari
- Moderation eksikligi

## P2 - Gym & Community Layer

### Amac

TrackMe'yi birebir coaching'den gym/club topluluklarina genisletmek.

### Basari Kriteri

Bir gym kendi uyelerini, coach'larini, feed'ini ve leaderboard'unu yonetebilir.

### Ozellikler

- Multi-gym
- Gym membership
- Gym coaches
- Gym feed
- Gym leaderboard
- Global leaderboard
- PR evidence videos

### Riskler

- Cok erken community karmasikligi
- Moderation yuku
- Yetki modelinin karmasiklasmasi

## P3 - AI, Growth & Monetization

### Amac

AI destekli uretkenlik, buyume kanallari ve gelir modelini gelistirmek.

### Basari Kriteri

Trainer daha hizli program uretir; athlete daha iyi yonlendirilir; platform surdurulebilir gelir uretir.

### Ozellikler

- AI program draft
- AI coaching suggestions
- Advanced analytics
- Public program discovery
- Optional Program Fork update flow
- Reklam altyapisi
- Rebranding
- Mobil app olgunlastirma

### Riskler

- AI'in erken ve kalitesiz cikmasi
- Marketplace'e erken kayma
- Reklamin core deneyimi bozmasi
