# Exercise Rules

- Exercise name is required.
- Exercise category is required in the MVP.
- Normalized exercise name must be unique.
- Slug must be unique.
- Measurement type is deferred until structured workout set logging.
- Measurement type controls allowed set log fields.
- Exercises used in workout history should be soft deleted instead of hard deleted.
- Admin manages official exercise library quality.
- Trainer users can create and maintain MVP exercise entries.
- Athlete users can read active exercise entries.
- Day 4 stores primary muscles, equipment, and instructions as editable text fields.
